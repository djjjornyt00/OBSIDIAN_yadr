
# Сетевой адаптер
==Сетевой адаптер== (Network Interface Card) — это аппаратное устройство, которое позволяет компьютеру подключаться к сети и обмениваться данными.
> Другие названия: сетевая карта, Ethernet-адаптер, LAN-адаптер

### задачи

- **преобразование данных**
— из параллельного формата ПК в последовательный для передачи по сети

- **формирование кадров**
— добавление MAC-адресов, контрольных сумм

- **управление доступом в среде**
— реализация протоколов CSMA/CD (Ethernet), CSMA/CA(Wi-Fi)

- **буферизация данных**
— временное хранилище кадров при приеме/передаче

### функции

- **физическая адресация**
Каждый адаптер имеет уникальный MAC-адрес(48 бит), прошитый производителем.

Используется для идентификации устройств в локальной сети.



- **передача/прием данных**
Преобразует биты в электрические/оптические сигналы
(==передача==) и обратно(==прием==).

Контролирует целостность с помощью CRC.



- **управление потоком**
Регулирует скорость передачи для предотвращения переполнения буфера получателя

# Характеристики сетевых адаптеров
## Скорость передачи
- 10 Мбит/сек
  Ethernet(устарело)
- 100 Мбит/сек
  Fast Ethernet
- 1 Гбит/сек
  Gigabit Ethernet(стандарт)
- 10 Гбит/сек → *25/40/100* Гбит/сек
  Для серверов и ЦОД

## Другие параметры

- **Тип интерфейса**
  PCI, PCIe, USB, M.2
- **Дуплекс**
  полудуплекс / полный дуплекс
- **Поддержка Wake-on-LAN(WoL)
  удаленное включение ПК
- **Поддержка VLAN, QoS**
  для корпоративных сетей

# Классификация по типу подключения
## Проводные адаптеры

•**Ethernet** (RJ-45)
 — стандартный разъём витой пары

•**Fiber** (SFP/SFP+)
 — для оптоволоконных сетей

| **Плюсы**        | **Минусы**               |
| ---------------- | ------------------------ |
| Стабильность     | Зависимость от кабеля    |
| Высокая скорость | Ограниченная мобильность |
| Низкая задержка  |                          |

## Беспроводные адаптеры

•Wi-Fi (802.11 a/b/g/n/ac/ax)
 — беспроводная ЛВС

•Bluetooth
 — короткие расстояния (до 100 м)

| Плюсы                  | Минусы        |
| ---------------------- | ------------- |
| Мобильность            | Помехи        |
| Простота развертывания | Скорость ниже |
|                        | Безопасность  |

# Классификация по форм-фактору

- **Встроенные (onboard)**
  Интегрированы в материнскую плату.
  Удобны для обычных ПК и ноутбуков.
- **PCI/PCIe карты**
  Устанавливаются в слот расширения.
  Используются для апгрейда или серверов.
- **USB-адаптеры**
  Подключаются через USB.
  Портативны, подходят для Wi-Fi, но медленнее PCIe.
- **M.2/NGFF**
  Компактный формат для ноутбуков и мини-ПК.
  Поддерживает Wi-Fi + Bluetooth.

# Драйвера сетевых адаптеров

**Драйвер** — программное обеспечение, которое позволяет операционной системе взаимодействовать с сетевым адаптером.
### задачи

**•Инициализация адаптера**
 — настройка при загрузке ОС

**•Управление передачей/приёмом**
 — формирование кадров, контроль ошибок

**•Обработка прерываний**
 — уведомление ОС о событиях (приём кадра, ошибка)

**•Настройка параметров**
 — скорость, дуплекс, MAC-адрес, Wake-on-LAN
